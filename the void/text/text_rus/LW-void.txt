Это подробный отчет об экспериментах, проведенных с пустотной жидкостью.
Вот часть первой записи:
"...кубический объем пустотной жидкости не имеет ограничений на количество вещества, которое она может растворить.<LINE>Однако, чем больше растворенной массы, тем больше времени потребуется для растворения большего количества вещества."
"Время, необходимое для растворения дополнительного вещества, равно квадрату количества вещества, которое оно уже израсходовало,<LINE>таким образом требуемое время для растворения будет стремиться к бесконечности".
"По этой причине необходимо установить дистилляционный насос, чтобы постоянно фильтровать жидкость в максимально чистом виде".
Затем в нем описывается несколько возможных конструкций фильтрующего насоса с различным уровнем эффективности.<LINE>Каждый из них проходит экспериментальную проверку и результаты регистрируются.
